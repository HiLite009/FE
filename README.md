## Project

### Stack
- React 19
- TypeScript
- Vite

### Install

1. Clone the repository
```bash
git clone https://github.com/HiLite009/FE
cd FE
```

2. Install dependencies
```bash
npm install
```

3. Run the development server
```bash
npm run dev
```

4. Open your browser and go to `http://localhost:5173/` to see the app in action.
